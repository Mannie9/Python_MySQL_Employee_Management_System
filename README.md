# Python MySQL Employee Management System

## 📌 Project Overview

This project demonstrates how Python can interact with a MySQL database to manage employee records.

The application allows users to enter employee information through a menu-driven interface and store the data directly into a MySQL database.

The goal of this project was to learn Python fundamentals while understanding how Python applications connect with relational databases.

# 🚀 Features

### Add Employee Records

Users can enter:

* Employee ID
* Employee Name
* Department
* Salary
* Bonus

### Store Data in MySQL

Employee data is inserted directly into the MySQL database using Python.

### Menu Driven Application

Interactive console-based employee management system.

### Dictionary-Based Data Storage

Employee information is managed using Python dictionaries before insertion into the database.

# 🔄 Project Workflow

1. Create MySQL database
2. Create employee table
3. Connect Python with MySQL
4. Accept employee information from user
5. Store data inside Python dictionary
6. Insert records into MySQL database
7. Commit changes
8. Close database connection
